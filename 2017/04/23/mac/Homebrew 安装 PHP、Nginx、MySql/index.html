Homebrew 安装 PHP、Nginx、MySql

Apr 23, 2017 | 技术杂谈 | Hits

安装 Mysql

1.查找mysql

2.查看版本信息
brew info mysql

3.执行安装启动

上面安装成功后提示的意思是说, 你安装的 MySql 没有密码, 执行mysql_secure_installation 进行设置密码, 其中会提示很多信息让你设置, 根据提示自己设置就行了。

4.安装完成进入mysql

mysql -uroot -p

安装PHP

1.超找php信息
brew search php

2.执行安装
brew install php

3.安装成功
php -i | grep php.ini 可以查找php.ini

安装nginx

方法同上brew install nginx

配置文件在
/usr/local/Cellar/nginx/
